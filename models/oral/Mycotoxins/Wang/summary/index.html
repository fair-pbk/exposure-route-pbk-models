    Notes


    DON PBK (Wang et al., human, compact): Stomach → Jejunum → Ileum → LI; absorption from each segment → Intestinal tissue → liver Liver perfusion-limited (venous) + intrinsic clearance; Blood ↔ Rest

    ODEs

    \(\frac{d[\mathtt{ASt}]}{dt} = - \mathit{ksto}\cdot \mathit{ASt}\)

    \(\frac{d[\mathtt{AJ}]}{dt} = \mathit{ksto}\cdot \mathit{ASt} - \mathit{kt1}\cdot \mathit{AJ} - \mathit{kabsJ}\cdot \mathit{CJ}\)

    \(\frac{d[\mathtt{AI}]}{dt} = \mathit{kt1}\cdot \mathit{AJ} - \mathit{kt2}\cdot \mathit{AI} - \mathit{kabsI}\cdot \mathit{CI}\)

    \(\frac{d[\mathtt{ALI}]}{dt} = \mathit{kt2}\cdot \mathit{AI} - \mathit{kabsLI}\cdot \mathit{CLI}\)

    \(\frac{d[\mathtt{AIT}]}{dt} = \mathit{kabsJ}\cdot \mathit{CJ} + \mathit{kabsI}\cdot \mathit{CI} + \mathit{kabsLI}\cdot \mathit{CLI} + \mathit{QIT}\cdot \mathit{CBlood} - \frac{\mathit{QIT}\cdot \mathit{CIT}}{\mathit{PIT}}\)

    \(\frac{d[\mathtt{ALiver}]}{dt} = \frac{\mathit{QIT}\cdot \mathit{CIT}}{\mathit{PIT}} + \mathit{QL}\cdot \mathit{CBlood} - \frac{(\mathit{QL}+\mathit{QIT})\cdot \mathit{CLiver}}{\mathit{PL}} - \mathit{CLint}\cdot \frac{\mathit{CLiver}}{\mathit{PL}}\)

    \(\frac{d[\mathtt{ABlood}]}{dt} = - \mathit{QIT}\cdot \mathit{CBlood} - \mathit{QL}\cdot \mathit{CBlood} + \frac{(\mathit{QL}+\mathit{QIT})\cdot \mathit{CLiver}}{\mathit{PL}} - \mathit{QRest}\cdot \mathit{CBlood} + \frac{\mathit{QRest}\cdot \mathit{CRest}}{\mathit{PRest}}\)

    \(\frac{d[\mathtt{ARest}]}{dt} = \mathit{QRest}\cdot \mathit{CBlood} - \frac{\mathit{QRest}\cdot \mathit{CRest}}{\mathit{PRest}}\)

    \(\frac{d[\mathtt{AMet}]}{dt} = \mathit{CLint}\cdot \frac{\mathit{CLiver}}{\mathit{PL}}\)
